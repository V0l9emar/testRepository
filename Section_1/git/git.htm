Git Teminal command

Otpravka

git pull : podqrujaet se izmeneniya v repozitorii s oblaka na komp

git init: inicalizaciya repositoiya

git status : proverka est li izmeneniya

git add . : primenit vse izmeneniya

git commit -m "firstCommit": zakomentirovat izmeneniya

git remote add origin https://..link.git : dobavit v oblako

git push -u origin master : vstavit v opredeleniye izmeneniya

Zaqruzka

git clone https://..link : zaqruzit iz oblaka
